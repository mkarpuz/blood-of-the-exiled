
# blood-of-the-exiled

This is a 3D browser game project based on Three.js.
The characters and assets will be created in Blender.

## Creation Steps

0. Research, Workflows & Architecture creation
1. Concepts and design: output Markdown files
2. Asset creation in Blender: output `.glb` exports
3. Three.js integration with the `.glb` files: actual codebase

## Current Implementation Step

The project is currently in the initial phase 0, where we are creating the workflow and doing research to find optimized ways forward. We are also creating pipelines for an efficient AI workflow.
